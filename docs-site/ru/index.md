# SoupaWhisper

SoupaWhisper — это невероятно быстрое десктопное приложение для голосовой диктовки, созданное на базе Tauri v2, React 18, TypeScript и Rust.

Приложение записывает звук с выбранного устройства ввода, транскрибирует его через облачный клиент, совместимый с Whisper (по умолчанию используется API Groq), при желании применяет словарь замен и постобработку через LLM, а затем выводит итоговый текст путем мгновенной автоматической печати (auto-typing) или копирования в буфер обмена.
