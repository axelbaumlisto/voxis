# Безопасность

## Обработка API-ключей

SoupaWhisper хранит ключи API (например, для Groq или OpenAI) в локальной базе данных SQLite (`config.db` в директории конфигурации платформы).

- Ключи **не** загружаются автоматически из переменных окружения.
- Приложение использует их для связи с выбранным провайдером транскрипции или LLM.
- **Никогда не добавляйте API-ключи в систему контроля версий (git).**

## Темы

Темы оверлея включают `theme.js`, который исполняется как доверенный JavaScript-код в контексте webview.
- **Предупреждение:** Выполняйте только тот код тем, которому вы доверяете. Загрузка сторонней темы похожа на установку расширения браузера.

## Хранение данных

Все данные (история, словари, неудачные аудиозаписи) хранятся локально на вашем компьютере.

- Включенный режим отладки (`debug=true`) сохраняет файлы WAV и JSONL в папку `debug/` без автоматической очистки, что может привести к увеличению потребления дискового пространства или раскрытию конфиденциальных записей. Отключайте этот режим при нормальном использовании.
